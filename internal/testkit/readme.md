# internal/testkit

Общие тестовые helpers.

Здесь можно держать код, который нужен нескольким тестовым пакетам:

- запуск тестовой PostgreSQL-схемы;
- MinIO/S3 round-trip helpers;
- загрузка golden-манифестов;
- assertions для ожидаемых рёбер golden-кластера;
- builders доменных объектов для unit-тестов.

Тестовая логика не должна попадать в production-пакеты ради удобства тестов. Если helper нужен только одному пакету, лучше держать его рядом с тестом.
