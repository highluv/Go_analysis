# internal/storage/minio

S3-compatible хранилище raw-манифестов.

В MVP реализация работает с MinIO из `deploy/infra`, но код лучше писать через S3-compatible API, чтобы не привязываться к конкретному локальному сервису.

Ответственность пакета:

- создать bucket при старте, если его нет;
- сохранить raw-манифест под детерминированным ключом;
- вернуть ключ, размер и hash для записи в PostgreSQL;
- достать объект обратно для round-trip теста.

Raw-объекты считаются immutable: существующий объект одного snapshot нельзя перезаписывать без явной ошибки.
